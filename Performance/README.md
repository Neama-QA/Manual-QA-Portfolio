# Web Performance Testing using Apache JMeter

## Project Overview
This project demonstrates performance testing for a web application using Apache JMeter.  
The test simulates user interaction with the system to evaluate response time, throughput, and system stability under load.
## Tool
Apache JMeter
## Test Type
Performance Testing

## Test Scenario
Simulating user requests to evaluate system performance.
## 🌟 Key Points
* End-to-end API workflow testing
* Dynamic user registration
* Token extraction and reuse for authenticated requests
* Reusable Module Controller for the Tasks API
* Clear assertions for status codes and response content

## 🚀 How to Run
**Step 1**
Open the test plan file in Apache JMeter
```text
TodoPerformanceTestPlan.jmx
```
**Step 2**
Run the Thread Group

**Step 3**
Inspect execution results using
```text
View Results Tree, View Aggregate Report
```
